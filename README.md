# Loon-Script

Loon 配置、插件与规则整理仓库。

## 目录结构

- `Config/`：Loon 配置文件（`.lcf`）
- `Plugin/`：Loon 插件（`.lpx`）
- `Rule/`：Loon 规则（`.lsr`）

## 使用建议

- tvOS 用户优先使用 `Loon_tvOS_Config_By_iKeLee_Emby.lcf`
- iPhone / iPad 用户可从 `Loon.lcf`、`Loon_CN.lcf`、`Loon_EN.lcf` 中选择
- 如果只需要单独功能，可直接引用 `Plugin/` 或 `Rule/` 中的原始链接
